Este site contem artigos, que estão separados nos seguintes grupos tematicos:
  • Cursos
  • Dicas
  • Projetos
  • Variedades
    • Existe uma segunda separação de artigos, por linguagens, ou frameworks, ou assuntos distintos, ou seja, você encontrará artigos divididos como nos exemplos abaixo:
  • Curso de Javascript (no grupo Cursos)
  • Dica de CSS (no grupo Dicas)
  • Projeto de Angular (no grupo Projetos)
  • Assuntos gerais (no grupo Variedades)
    • Então um artigo pode ser, por exemplo:
  • Aula de um curso
  • Dica sobre um framework
  • Comentário de um projeto
  • Sobre assuntos gerais
    • Ou seja, todo o conteudo do site está contido em Artigos.
    Navegue no site, é bastante intuitivo.
    Sobre os artigos:
    As indicam o nível considerado para o Artigo, podendo ser:
  • Iniciante
  • Intermediario
  • Avançado
    • A palavra NEW indica que o Artigo é novo no site, uma publicação considerada recente.
