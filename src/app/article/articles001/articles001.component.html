Download do Node JS

Acesse o site nodejs.org e na página inicial já haverá o link para o download.

Você pode também acessar o site nodejs.org/pt-br que é apresentando em Portugues do Brasil.

Prefira a versão LTS, que, embora não seja a última, é a versão estável.

A versão atual disponível no site é a 14.15.1 LTS.

Clique no link e faça o download do instalador.

Instalação do Node JS

Clique no arquivo baixado para iniciar a instalação.

A instalação é simples, basta somente seguir com a instalação padrão, clicando sempre em Next.

Finalizada a instalação, clique em concluir

Validação da instalação do Node JS

Abra um terminal de comandos, que pode ser o terminal do próprio Windows.

Se preferir o terminal do Windows, digite cmd no campo de pesquisa da barra de tarefas e seleciono o Prompt de Comando.

Para validar a instalação do Node JS execute o comando:

node --version

Deverá ser exibida a versão do Node JS instalada.

Sobre o NPM integrado ao Node JS

Na instalação do Node JS também é instalado o NPM, que lhe possibilitará acesso a uma biblioteca com uma infinidade de ferramentas que poderão ser integradas ao seu projeto, para os mais diversos usos.

Para validar a instalação do NPM execute o comando:

npm --version

Vale lembrar que a utilização do NPM é independente do NodeJS.

Muitos desenvolvedores instalam o Node JS somente com o intuito de utilizar o NPM.
